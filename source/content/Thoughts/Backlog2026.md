
SMBU:
1. Time-Varying:
	1. (опубликовано в 2026) Обучение Time-Invariant модели с семплингом time-invariant наблюдений из цепочки. Предложены модели классификации с временной обусловленностью (TCNN). Прогноз для цепочке через взвешенную сумму прогнозов отдельных наблюдений (3 схемы). Сравнение с CoxTV, CoxPH, DynamicDeepHit, CoxPH+sampling, TCNN на наборе Backblaze.
	2. (не доделано) Исследование масштабируемости обучения (с 1 наблюдения = time-invariant до N = time-varying). Есть прирост, но нужны хорошие исследования и обоснования.
	3. (нет результатов) Модификация TCNN или смена формата прогноза (нужно фиксировать шкалу для воспроизводимости)
	4. (нет результатов) Анализ потока действий пользователя (e-com) на маркетплейсах (известен id товара и действие пользователя). Необходимо спрогнозировать время до следующего события определенного типа (например, время до следующей покупки). Для решения на основе потока действий пользователя строится embedding по графу товаров и одна Time-invariant модель.
	5. (нет результатов, перспективно) По потоку time-varying наблюдений (с учетом времени сбора) стоить один embedding, на котором обучать time-invariant модель.
	6. (нет результатов, перспективно) Анализ потока логов событий (Event Viewer) для прогнозирования следующего терминального. Кроме модели прогнозирования нужно иметь интерпретацию правил (и выделение EVENT_ID "предвестников деградации", лучше по разным типам отказа). 
2. Competing risks:
	1. (опубликовано) Построение модели конкурирующих рисков через ансамбль OvR по 3м схемам (тип k vs остальные как цензурированные, тип k vs только цензурированные, тип k vs смесь). Лучшие результаты показала вторая схема (где k vs только цензурированные). Для построения единого прогноза используется метамодель (RF) с выбором наиболее вероятного события и взвешиванием разных событий. На наборе Freddie Mac получилось обойти существующие модели (DeepHit, DeepSurv, AJ) по классическим метрикам SA.
	2. (не опубликовано) Модификация метрик анализа выживаемости для оценки конкурирующих рисков (учет не одного общего прогноза, а всех S(t) по всем рискам). Разработан IBS (одна общая оценка).
	3. (нет результатов, перспективно) Модификация критерия Log-rank для обработки конкурирующих рисков. На этапе разбиения строится вектор log-rank оценок по всем типам событий. По агрегату (максимум, минимум, среднее) ищется лучшее разбиение. В листе дерева строится множество непараметрических моделей по каждому типу риска, возвращается прогноз по листу.
	4. (нет результатов, перспективно) Разложение исходного распределения времени терминальных событий на множество унимодальных (по GMM) -> построение модели конкурирующих рисков (пункт 1) -> построение общего прогноза -> сравнение с классическими терминальными (только надо дать интересную интерпретацию/картинку)
3. Модели с повторными событиями:
	1. (нет результатов, перспективно) исследование качество существующих моделей RecurrentEvents (с разбиением start/stop и использующих номер события k) и сравнение с предложенными time-invariant моделями "следующего события" (на обычных признаках +- k +- RFM). Анализ зависимости качества от номера события.
4. Сравнение SA с другими задачами (поиск места моделей выживаемости на рынке)
	1. (нет результатов, но скоро будут) Survey benchmark сравнение моделей классификации, регрессии и SA на разных наборах данных (и разным метрикам каждой из задач). Кроме сравнения нужно понятно отобразить модели на плоскости (как квадрат гартнера). Плюс, можно сделать RAG-систему рекомендации лучшей модели под задачу пользователя.
	2. (нет результатов) Расширение методов прогнозирования оттока клиентов: замена моделей классификации на SA, более точный расчет digital-метрик (на примере CLV: замена непараметрической S(t) на S(t|X) индивидуальную). Исследование влияния "вмешательства" (масштибирование S(t|X))
5. Обработка мультимодальности в деревьях и статистических моделях:
	1. (нет результатов) Модификация CoxPH/AFT с базовой функцией на виртуальных событиях
	2. (нет результатов) Генерация событий (SMOTE): может работать как виртуальные события в деревьях (схожее признаковое пространство в листах). Генерация событий через семплинг промежуточных событий (по медиане/среднему случайных пар).
	3. (не опубликовано) Добавление в критерий разбиения log-rank устойчивости к дисбалансу (есть наработки в разбиении узлов, но не проверено качество)
	4. (не опубликовано) Детектирование мультимодальности распределения с помощью AUPRC (уже известна хорошая корреляция с расстоянием между модами). Перспективная работа, но нужно большое экспериментальное исследование с выводами и объяснением.
